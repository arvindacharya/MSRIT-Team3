A webpage which reads the data from the JSON file provided and shows it in the form a graph.

You have the option to choose from 2 kinds of graphs - candlestick and bar graph.

You can choose to see the data from 10 different company symbols.

How to run the solution- 
1. Open index.html 
2. Toggle between the different options available
